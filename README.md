# Operations & Development Projects

## Food Retail Traceability App
**Automated Ingredient Tracking System**

### Problem
Manual ingredient tracking consuming 8-12 hours/week for food retail business with 50+ daily transactions.

### Solution
Android mobile app with real-time data synchronization to eliminate manual data entry and improve traceability compliance.

### Tech Stack
- Android Studio
- Google Firebase
- Claude Code (AI-assisted development)

### Key Features
- Real-time ingredient lot tracking
- Automated data entry and validation
- Cloud-based storage and reporting
- Offline-first architecture

### Impact
- Eliminated 8-12 hours/week of manual work
- Improved regulatory compliance
- Real-time visibility into ingredient traceability
